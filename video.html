Video Compression Project

Project Description

The goal of this project was to demonstrate a key prinicple in multimedia transmission; compression. Techniques demonstrated are;

  • Pulse Code Modulation
  • Differential Pulse Code Modulation with different levels of quantization (2 bit, 4 bit etc)
  • Discrete Cosine transform
  • Differential Pulse Code Modulation on a video compressed by Discrete Cosine Transform

Skills and Software Used

  • MATLAB
